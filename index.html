Lists in HTML

There are 3 main types of lists: unorder (bullet points) and ordered (numbers, letters, lathin number,...) and description list (list with a description) We use the <ol> tags for ordered or <ul> for unorderd. The description Lists need the <dl> and also the <dt> for the term and <dd> for the description itself. For each bullet point we now make a <$i><$/li> tag, in the tags above.

Lists in CSS

ul.b { list-style-image: url('my-bullet.png'); } list-style-image: imports the picture of a URL and uses it for the bullet point

ul.d { list-style-position: inside; } list-style-positon says the position of the point and it can be inside or outside.

Box-Model

inline elements: always start in a new line and have top and bottom margins.
margin: it clears an area outside the border. you find it outside the border
padding: it clears an area around the content and its transparent, while the margin clears the outside of the border element.

  • steht für graphical interchange format, häufig animiert in einer Endlosschleife

  • wird von Windows für icons verwende, in einer Datei mehrer Bilder, 256×256 Pixel, 16 oder mehr Farben mögich

  • steht für scalable vector graphics, Vektrografik, mathematisch definierte Bilder, die sich aus Punkten
  • und Linien zusammensetzten, ideal für Logos und komplexe Online grafiken, sind in XML geschrieben, so kann Google
  • Bilder über den Quellcode suchen

    • mpg/mpa

  • von der Firma mpeg, für Videodateien, unterstützt Animationen und enfache Bilder, kompressierbar

    • Gradians

    Mit Gradients kann man Farbübergänge zwischen mehren Farben erstellen, Dazu gibt es 3 Arten:

    1. Lineare Gradianten(von oben nach unten, links nach rechts, usw.)
    2. Radial Gradiaten(geht von Center aus)
    3. Conic Gradianaten(rotiert um das Cnenter herum)

    background-image: linear-gradient(direction, color-stop1, color-stop2, ...); -> linear

    Mehrere Variaten

    #grad {
    background-image: linear-gradient(red, yellow);
    }


    #grad {
    background-image: linear-gradient(to right, red , yellow);
    }


    #grad {
    background-image: linear-gradient(180deg, red, yellow);
    }


    background-image: radial-gradient(shape size at position, start-color, ..., last-color); ->radial

    #grad {
    background-image: radial-gradient(red, yellow, green);
    }

    #grad {
    background-image: radial-gradient(red 5%, yellow 15%, green 60%);
    }

    #grad {
    background-image: radial-gradient(circle, red, yellow, green);
    }

    Der size-Operator ist entweder: closest-side farthest-side closest-corner farthest-corner

    #grad {
    background-image: repeating-radial-gradient(red, yellow 10%, green 15%);
    }

    background-image: conic-gradient([from angle] [at position,] color [degree], color [degree], ...); -> conic

    #grad { background-image: conic-gradient(red, yellow, green); }

    #grad { background-image: conic-gradient(red 45deg, yellow 90deg, green 210deg); }

    #grad { background-image: conic-gradient(red 0deg, red 90deg, yellow 90deg, yellow 180deg, green 180deg, green 270deg, blue 270deg); border-radius: 50%; }

    Transformation

    Mit dem Transform Argument stehen einem mehere Methoden zur Verfügung:

    1. translate: bewegt das Objekt von einer Stelle an die neuen Kordinaten
    2. rotate: dreht das Objekt
    3. scale/X/Y: vergrößert/verkleinert das Objekt in beide/X/Y Richtung
    4. skew/X/Y: verdreht das Objekt in beide/X/Y Richtung
    5. matrix: Die Marix Funktion vereint alle der transform Elemete und braucht 6 Parameter

    div {
    transform: translate(50px, 100px);}

    Animationen

    Mit CSS-Animationen lässt sich der Style von Elementen ändern, dazu wird das Ausgangsframe mit @keyframe gekennzeichnet

    @keyframes example {
    from {background-color: red;}
    to {background-color: yellow;}
    }
    /* The element to apply the animation to */
    div {
    width: 100px;
    height: 100px;
    background-color: red;
    animation-name: example;
    animation-duration: 4s;
    }

    Mit Prozent Angaben statt den from, to statements, lassen sich mehrere Effekte machen

    Mit animation-delay und animation-duration können Dauer und delay bestimmt werden

    Mit animation-iteration-count lässt sich angeben wie oft eine Animation durchgeführt wird, mit infite wird es für imemr durchgeführt.

    Transition

    Mit Transition lassen sich in bestimmten Zeiträume Werte von Objekten ändern

    Um diese zu Benutzen muss man die Dauer und die Eigenschaft angeben

    div {
    width: 100px;
    height: 100px;
    background: red;
    transition: width (Eigenschaft) 2s(Dauer);
    }

    Mit hover kann dies dann ausgenützt werden

    ease- spezifiziert einen Übergangseffekt mit einem langsamen Start, dann schnell, dann langsam enden (dies ist die Vorgabe) linear- gibt einen Übergangseffekt mit der gleichen Geschwindigkeit von Anfang bis Ende an ease-in- legt einen Übergangseffekt mit langsamem Start fest ease-out- legt einen Übergangseffekt mit langsamem Ende fest ease-in-out- legt einen Übergangseffekt mit langsamem Start und Ende fest cubic-bezier(n,n,n,n)- lässt Sie Ihre eigenen Werte in einer kubischen Bezier-Funktion definieren

    Mit transition-delay lässt sich eine Verzögerung erstellen
