Lists in HTML

There are 3 main types of lists: unorder (bullet points) and ordered (numbers, letters, lathin number,...) and description list (list with a description) We use the <$ol><$/ol> tags for ordered or <$ul><$/ul> for unorderd. The description Lists need the <$dl><$/dl> and also the <&dt><&/dt> for the term and <$dd><$/dd> for the description itself. For each bullet point we now make a <$i><$/li> tag, in the tags above.

Lists in CSS

ul.a { list-style-type:square; } list-style-type: give the type of bullet point, like if its a square, cyrcle, roman literal and so on

ul.b { list-style-image: url('my-bullet.png'); } list-style-image: imports the picture of a URL and uses it for the bullet point

ul.d { list-style-position: inside; } list-style-positon says the position of the point and it can be inside or outside.

Box-Model

block elements: don’t start from a new line and don’t have top and bottom margins
inline elements: always start in a new line and have top and bottom margins.
margin: it clears an area outside the border. you find it outside the border
padding: it clears an area around the content and its transparent, while the margin clears the outside of the border element.
